# Fukidashi System for VRChat Avatars 3.0

VRChat などでアバターに仕込むことが出来るメッセージカードシステムです。  
テクスチャーに設定したメッセージを最大16個まで切り替え表示することが出来ます。

なお、プロジェクト本体は GitHub にて公開されており、知識があれば誰でも無料で使用およびシステムの修正が可能です。
https://github.com/mika-f/VRChat-FukidashiSystem


## 注意事項

不具合などがあるかもしれません。ご了承ください。
なお、以下の環境にて動作を確認しております。

* Unity 2018.4.20f1
* VRCSDK3-AVATAR 2020.08.06.16.30
* VRCSDK3-WORLD 2020.08.07.18.18 (インポートの必要ありません)
* VRChat 2020.3.2p1, Build 970


## GitHub 版との違いについて

BOOTH ではビルド済みの UnityPackage と説明書を付けたパッケージを配布しています。
GitHub 版でもほぼ同様のものは作成可能ですが、サポートなどは行いません (自己解決出来る人向けです)。
