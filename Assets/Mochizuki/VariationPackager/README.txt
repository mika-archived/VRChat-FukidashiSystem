# Variation Packager for Unity

複数 (例: エディション違いなど) の UnityPackage および Zip アーカイブをボタン1つで作成できるようにする Unity エディター拡張です。  
ビルド構成情報を Unity のシーンファイルもしくは JSON 形式で保存することで、次回以降も使い回すことが出来ます。
簡易的なビルドデバッガーとプリ・ポストプロセス機能付きです。


## GitHub 版との違いについて

BOOTH ではビルド済みの UnityPackage と説明書を付けたパッケージを配布しています。
GitHub 版でもほぼ同様のものは作成可能ですが、サポートなどは行いません (自己解決出来る人向けです)。
